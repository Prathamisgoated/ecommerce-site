# ShopWave — React E-Commerce Website

A modern React-based e-commerce website featuring a responsive design, product showcase, category sections, shopping cart functionality, and smooth user experience.

## Project Structure

```text
src/
│
├── components/
│   ├── CategoryShowcase.jsx
│   ├── CategoryShowcase.css
│   ├── Features.jsx
│   ├── Features.css
│   ├── Footer.jsx
│   ├── Footer.css
│   ├── Hero.jsx
│   ├── Hero.css
│   ├── Marquee.css
│   ├── Navbar.jsx
│   ├── Navbar.css
│   ├── ProductCard.jsx
│   ├── ProductCard.css
│   ├── ProductGrid.jsx
│   ├── ProductGrid.css
│   ├── Reviews.jsx
│   ├── Reviews.css
│   ├── Toast.jsx
│   ├── Toast.css
│   └── products.jsx
│
├── App.js
├── App.css
└── index.js

Root Files
│
├── package.json
├── package-lock.json
├── .gitignore
└── README.md
```

## Features

- Responsive Navigation Bar
- Hero Landing Section
- Product Grid
- Product Cards
- Category Showcase
- Customer Reviews Section
- Feature Highlights
- Toast Notifications
- Responsive Footer
- Modern UI Design

## Technologies Used

- React.js
- JavaScript (ES6+)
- CSS3
- HTML5

## Installation

```bash
npm install
npm start
```
